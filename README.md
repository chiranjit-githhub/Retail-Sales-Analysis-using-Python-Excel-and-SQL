# 🛒 Retail-Sales-Analysis-using-Python-Excel-and-SQL

## 📌 Project Overview

The **Retail Store Sales Analysis Dashboard** is an end-to-end data analytics project that analyzes retail transaction data to uncover business insights. The project uses **Python** for data cleaning and visualization, **MySQL** for business analysis using SQL queries, and **Microsoft Excel** for building an interactive dashboard with Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

This project helps businesses monitor sales performance, identify top-selling products, evaluate store and cashier performance, and understand customer payment preferences.

---

## 🎯 Objectives

- Analyze retail sales data to identify business trends.
- Perform data cleaning and exploratory data analysis (EDA).
- Solve business problems using SQL queries.
- Build an interactive Excel dashboard for decision-making.

- Performed data preprocessing

### 3. Exploratory Data Analysis (Python)
- Sales by Product
